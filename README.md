# BobbySignSearch
A Python 3.13 script for finding a sign with the desired text in the Bobby mod cache.

Usage:
1. Launch main.py or BobbySignSearch.exe
2. Enter the number of CPU threads
3. Enter the path to the server cache folder
4. Enter the text you want to find.
5. Wait...
6. Check result in bobby_search_report.txt

###### Tested only in 1.21.8
